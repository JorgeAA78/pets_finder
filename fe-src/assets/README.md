# Assets

Carpeta para recursos estáticos del proyecto.

## Estructura

```
assets/
├── images/      # Imágenes del proyecto (logos, iconos, placeholders)
└── README.md
```

## Uso

Para importar imágenes en los componentes:

```typescript
import logo from './assets/images/logo.png';
```

O en HTML:

```html
<img src="./assets/images/logo.png" alt="Logo">
```
